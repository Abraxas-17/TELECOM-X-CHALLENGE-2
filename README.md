📊# TELECOM-X-CHALLENGE-2
CHALLENGE TELECOM X 2
📌Descripción

Este proyecto analiza los factores que influyen en la cancelación de clientes (churn) en una empresa de telecomunicaciones. A través de técnicas de análisis de datos y modelos de machine learning, se busca identificar patrones en el comportamiento de los clientes y predecir qué usuarios tienen mayor probabilidad de cancelar el servicio.

🎯Objetivo

Identificar las variables que más influyen en la cancelación de clientes y desarrollar modelos predictivos que permitan anticipar este comportamiento para apoyar estrategias de retención.

⚙️Metodología

El análisis se desarrolló en varias etapas:

Limpieza y preparación de datos
Transformación de variables categóricas mediante one-hot encoding
Análisis exploratorio de datos
División del dataset en entrenamiento y prueba
Entrenamiento de modelos de clasificación
Evaluación de desempeño mediante métricas de clasificación
Modelos utilizados
Regresión Logística
Árbol de Decisión

Estos modelos permitieron comparar diferentes enfoques para la predicción del churn.

Principales hallazgos

Se identificaron varios factores asociados con una mayor probabilidad de cancelación:

Clientes con menor tiempo de permanencia

Contratos de tipo mes a mes

Pagos mediante Electronic Check

Mayores cargos mensuales

Tipo de servicio de internet

📌Conclusión

Los modelos desarrollados permiten identificar clientes con alto riesgo de cancelación, lo que puede ayudar a la empresa a implementar estrategias de retención más efectivas y mejorar la satisfacción del cliente.
